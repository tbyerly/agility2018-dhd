Class 2: Building the F5 Fabric
===============================

Welcome to the Building the F5 Fabric lab session at F5 Agility |year|.
These labs are intended to guide you through creating basic Device
Service Cluster (DSC) using the GUI, TMSH and in an automated fashion.

This guide is intended to complement lecture material provided during the
Agility event as well as a reference guide that can be referred to after the class.
