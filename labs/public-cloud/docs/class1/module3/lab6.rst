Destroy environment
-------------------

1. Run the lab-cleanup script.

.. code-block:: bash

   lab-cleanup

2. Followed by "terraform destroy". Confirm destroy with 'yes' when prompted.

.. code-block:: bash

   terraform destroy

.. image:: ./images/lab-cleanup.png
  :scale: 50%

3. The end.
